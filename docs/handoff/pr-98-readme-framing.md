# Handoff — PR 98: README framing — link replacement as one feature

## Что сделано

Реализован issue #97 — минорная правка framing'а в `README.md` (3 места),
чтобы замена ссылок описывалась как одна из фич, а не как единственное
назначение софта.

### Tagline (строка 3)

- Было: «Редактор постов Telegram-каналов — замена ссылок через бота-админа»
- Стало: «Платформа для управления Telegram-каналами через бота-админа»

### `## Описание` (строки 5-29)

- Первый абзац переписан: telesoft — инструмент для управления
  Telegram-каналами через бота-админа. На данный момент реализована одна
  фича — массовая замена ссылок в постах канала с сохранением форматирования.
  Дальше — описание workflow (как было).
- Workflow, таблица 3 режимов, «Сохранение форматирования», чекбокс
  link_preview — без изменений (та же фича, тот же контекст).

### `## Возможности` (строки 51-63)

- Первый буллет: **Управление каналами**: список, добавление, редактирование,
  удаление, активация/деактивация (архивирование) — отражает PR#96
  (action menu + edit + is_active toggle).
- Второй буллет: **Замена ссылок**: массовая замена ссылок в постах канала
  с сохранением форматирования (3 режима — Simple, Library, Advanced) —
  как одна из фич.
- Остальные буллеты (Pattern library, Preview-confirm-run, link_preview,
  full_replace, форматирование, Entity URLs, Задачи, Mobile responsive) —
  без изменений.

### Не тронуто

- Архитектура, Стек, Требования, Быстрый старт, Локальная разработка,
  Переменные окружения, Настройка бота, Тестирование, Структура проекта,
  Деплой, Ограничения, Лицензия — без изменений.

## Почему

Проект растёт: PR#96 добавил action menu + edit + is_active toggle на
каналах — управление каналами теперь полноценное, не только список для
замены ссылок. В будущем планируются другие фичи. Framing README должен
отражать это: telesoft — платформа для управления каналами, замена ссылок —
одна из (пока единственная) фич.

Старый framing («Редактор постов Telegram-каналов — замена ссылок»)
позиционировал софт как инструмент ТОЛЬКО для замены ссылок, что не
соответствует текущему состоянию и блокирует расширение.
