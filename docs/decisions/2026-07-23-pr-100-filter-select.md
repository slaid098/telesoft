# ADR PR#100 — Channels filter toggle to select dropdown

## Статус

Accepted

## Контекст

Toggle-кнопка «Все каналы» / «Только активные» над формой «Добавить
канал» на странице `/channels` читалась как навигация (переход к списку
всех каналов) или как отмена формы при её открытии, а не как фильтр
списка. Это вводило пользователя в заблуждение и создавало визуальный
шум: при открытой форме toggle оставался видимым и конкурировал за
внимание с кнопкой «Закрыть».

## Решение

Заменить toggle-кнопку на `<select>` dropdown с `<label>Фильтр</label>`
(паттерн 1-в-1 как на `/jobs` из PR#92). Состояние `showInactive: boolean`
заменено на `channelFilter: "active" | "all"`. Select обёрнут в
`{#if !showForm}...{/if}` — скрывается при открытии формы «Добавить
канал», убирая визуальный шум и неоднозначность.

## Альтернативы

- **Переименовать toggle**: изменить текст кнопки, чтобы снять
  неоднозначность. Не решает визуальный шум при открытой форме и
  оставляет toggle-паттерн, который слабее select для фильтрации.
- **Оставить как есть** (status quo): toggle над формой сохраняется.
  Неоднозначность «навигация vs фильтр» и шум остаются — issue #99 не
  закрыт.
