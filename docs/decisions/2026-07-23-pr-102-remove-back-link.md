# ADR PR#102 — Remove back to channels link on detail page

## Статус

Accepted

## Контекст

Ссылка «Назад к каналам» в шапке детальной страницы `/channels/[id]`
дублировала верхний nav bar (из PR #74), который уже содержит пункт
«Каналы» → `/channels` и виден на всех страницах. На одном экране
присутствовало два способа уйти на список каналов — избыточно и
создавало визуальный шум.

## Решение

Убрать ссылку «Назад к каналам» из `web/src/routes/channels/[id]/+page.svelte`.
Nav bar остаётся единственным путём возврата к списку каналов.

## Альтернативы

- **Оставить ссылку** (status quo): дублирование nav bar сохраняется,
  визуальный шум остаётся — issue #101 не закрыт.
- **Убрать top nav**: удалить пункт «Каналы» из nav bar, оставив только
  ссылку на детальной странице. Лишает единой навигации на остальных
  страницах — регресс UX.
