# ADR PR#106 — Remove redundant open button on mobile cards

## Статус

Accepted

## Контекст

Кнопка «Открыть» на mobile-карточках каналов дублировала
кликабельный заголовок-ссылку, который уже покрывал навигацию на
страницу канала. На desktop аналогичной кнопки нет — переход
происходит по клику на название канала в таблице. Расхождение
между mobile и desktop по способу навигации + избыточный UI-элемент.

## Решение

Убрать кнопку «Открыть» с mobile-карточек. Оставить клик по
названию канала (заголовок-ссылка) как единственный способ
навигации — консистентно с desktop.

## Альтернативы

- **Добавить кнопку «Открыть» на desktop**: привело бы к
  дублированию на desktop (название + кнопка), усугубляет проблему
  вместо решения.
- **Оставить обе кнопки на mobile**: сохраняет избыточность и
  расхождение с desktop, issue #105 не закрыт.
