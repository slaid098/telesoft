# telesoft

Telegram channel post editor — replace links in channel posts via a bot admin.

## Stack

- **Backend**: Python 3.12+, FastAPI, aiosqlite (SQLite), Telethon (bot mode), uv
- **Frontend**: SvelteKit 2 + Svelte 5 + TypeScript, Biome, Vitest, Knip
- **Deploy**: Docker Compose

## Status

Pre-alpha. See `docs/` for project map and handoffs.
